# Reports

This directory contains generated analysis outputs and model evaluation artifacts.

## Directory Layout

```
reports/
├── figures/        ← All generated charts and plots (PNG, 150 DPI)
│   ├── eda_sensor_correlation.png
│   ├── eda_rul_distribution.png
│   ├── shap_summary_fd001.png
│   ├── shap_waterfall_engine42.png
│   ├── roc_curve.png
│   └── rul_prediction_vs_actual.png
└── README.md       ← This file
```

## Generating Reports

All figures are generated by:

```bash
make evaluate          # Runs src/visualization/visualize.py
```

Or from the evaluation notebook:

```bash
jupyter nbconvert --to notebook --execute notebooks/03_model_evaluation.ipynb
```

> Figures in `reports/figures/` are excluded from git (`.gitignore`). To share results, attach them to a GitHub Release or reference an external storage URL.
